# xlx-cli
A simple CLI 

#### Installation 
Prerequisites: Node.js (>=6.x, 8.x preferred), npm version 3+ and Git.

```
$ npm install -g xlx-cli
```

#### Usage

```
$ xlx-cli init <template-name> <project-name>
```
add template

```
$ xlx-cli add
```

Example:

```
$ xlx-cli init template my-project
```
